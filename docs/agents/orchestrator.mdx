# Orchestrator Agent

The Orchestrator is the central workflow engine that coordinates all other agents in the Eco ecosystem.

## Responsibilities

- **Task Distribution**: Assigns tasks to appropriate specialized agents
- **Dependency Management**: Ensures agents execute in the correct order
- **Resource Allocation**: Manages computational resources across agents
- **Quality Control**: Coordinates with Assist for validation
- **Provenance Chain**: Maintains cryptographic proof of all operations

## Workflow Stages

1. Request intake and analysis
2. Agent selection and task breakdown
3. Execution coordination
4. Quality validation
5. Provenance sealing and commitment
