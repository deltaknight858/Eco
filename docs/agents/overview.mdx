# Agents Overview

Agents are specialized workers in the Eco ecosystem, each designed for specific tasks while maintaining provenance throughout their operations.

## Core Principles

- **Specialization**: Each agent focuses on a specific domain
- **Provenance**: All operations are tracked and sealed
- **Coordination**: Agents work together through the Orchestrator
- **Quality**: Assist agent validates all outputs

## Available Agents

- [Assist Agent](./assist.mdx) - Front-end companion
- [Orchestrator](./orchestrator.mdx) - Workflow coordination
- [Provenance Agent](./provenance.mdx) - Cryptographic sealing
