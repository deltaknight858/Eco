# Getting Started

Welcome to **Eco** — an app‑creation ecosystem.  
This guide will help you set up your environment and run your first Power Page.

- Clone the repo
- Install dependencies with `pnpm install`
- Run the Shell: `pnpm dev --filter @eco/shell`
- Open `http://localhost:3000`
