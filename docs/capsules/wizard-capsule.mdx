# Wizard Capsule

The Wizard Capsule provides guided, step-by-step interfaces for complex workflows within the Eco ecosystem.

## Features

- **Progressive Disclosure**: Information revealed as needed
- **State Management**: Maintains user progress through multi-step processes
- **Validation**: Real-time validation at each step
- **Provenance Integration**: Tracks user decisions and inputs
- **Responsive Design**: Adapts to different screen sizes and contexts

## Use Cases

- Initial setup and onboarding
- Complex configuration wizards
- Multi-step data collection forms
- Guided troubleshooting flows
- Feature discovery and tutorials

## Integration

Wizard Capsules integrate seamlessly with the Shell framework and can invoke other agents through the Orchestrator as needed for validation and processing.
