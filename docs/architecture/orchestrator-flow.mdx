# Orchestrator Flow

The Orchestrator is the workflow engine that coordinates all agents in the Eco ecosystem.

## Flow Stages

1. **Request Reception**: Receives tasks from Power Pages or external sources
2. **Agent Assignment**: Determines which agents are needed for the task
3. **Coordination**: Manages agent execution order and dependencies
4. **Quality Assurance**: Validates outputs through Assist agent
5. **Provenance Sealing**: Applies cryptographic proof of work completion

## Agent Types

- **Assist**: Front-end companion and validation
- **OCR**: Optical character recognition
- **Compiler**: Code compilation and transformation
- **Tracker**: Resource and activity monitoring
- **Marketplace**: Plugin management and distribution
