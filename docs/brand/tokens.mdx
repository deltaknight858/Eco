# Design Tokens

The core design tokens define the visual foundation of the Eco ecosystem.

## Color Palette

### Brand Colors
- **Cyan**: `#00F5FF` - Primary brand color, used for provenance and key actions
- **Violet**: `#9D4DFF` - Secondary brand color, used for navigation and structure
- **Magenta**: `#FF2ED1` - Accent color, used for highlights and interactive elements
- **Gold**: `#FFD700` - Special accent, used for premium features and achievements

### Surface Colors
- **Deep Space**: `#0A0A0F` - Primary background
- **Panel**: `#14141A` - Secondary background for panels
- **Dark Glass**: `rgba(10, 10, 15, 0.7)` - Translucent overlays
- **Light Glass**: `rgba(245, 245, 250, 0.6)` - Light translucent elements
- **Frosted**: `rgba(255, 255, 255, 0.08)` - Subtle glass effects

### Text Colors
- **Primary**: `#F5F5F7` - Main text color
- **Muted**: `#9CA3AF` - Secondary text color
- **Provenance**: `#00F5FF` - Provenance-related text

## Typography

### Font Families
- **Sans**: `'Inter', system-ui, sans-serif` - Primary interface font
- **Mono**: `'JetBrains Mono', monospace` - Code and provenance font

## Effects

### Glow Effects
- **Cyan Glow**: `0 0 24px rgba(0, 245, 255, 0.25)`
- **Violet Glow**: `0 0 24px rgba(157, 77, 255, 0.25)`
- **Magenta Glow**: `0 0 24px rgba(255, 46, 209, 0.25)`
