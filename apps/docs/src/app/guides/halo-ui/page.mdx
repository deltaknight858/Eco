## ğŸ“š Component Library

### Core Components

- **HaloButton** - Interactive buttons with multiple variants
  - `primary` - Main call-to-action with orange gradient
  - `secondary` - Secondary actions with purple gradient  
  - `outline` - Outlined style with neon border
  - `ghost` - Minimal style for subtle interactions

- **HaloCard** - Content containers with glassmorphism effects
  - Backdrop blur and subtle shadows
  - Responsive padding and spacing
  - Perfect for highlighting content sections

- **HaloAlert** - Contextual message components
  - `info`, `success`, `warning`, `error` types
  - Consistent styling with appropriate color coding
  - Clear typography and spacing

- **HaloBadge** - Small indicator components
  - `default`, `premium`, `enterprise` variants
  - Tier-specific styling and colors
  - Compact design for status indication

## ğŸ¯ Design Principles

### Provenance-Aware Design
Every component embodies the **provenance** philosophy:
- **Transparency**: Visual clarity and honest communication
- **Authenticity**: Genuine interactions without deceptive patterns
- **Trust**: Reliable, predictable behavior across all components

### Performance-First Architecture
- **React 18 Optimized**: Leverages concurrent features and Suspense
- **TypeScript Native**: Full type safety and excellent DX
- **Tree-Shakeable**: Import only what you need
- **Minimal Bundle Impact**: Optimized for fast loading

### Developer Experience
- **Intuitive APIs**: Props that make sense and follow conventions
- **Comprehensive TypeScript**: IntelliSense and error prevention
- **Flexible Composition**: Components work well together
- **Easy Customization**: Override styles without fighting the framework

### Custom Styling

All components accept a `className` prop for custom styling:

```tsx

  variant="outline" 
  className="w-full md:w-auto hover:shadow-2xl"

  Custom Styled Button

```

### Theme Integration

Components are designed to work with the Eco design token system:

```css
/* Your app's CSS */
@import '@eco/core-tokens/tokens.css';

.my-custom-component {
  background: var(--eco-dark);
  color: var(--eco-text);
  border: 1px solid var(--eco-accent);
}
```
